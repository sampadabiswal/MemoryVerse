# 🌌 MemoryVerse

> **A Cinematic Birthday Memory Website**

MemoryVerse is a personalized interactive website created as a birthday surprise. It presents cherished memories in the form of seven cinematic chapters with photos, music, and smooth navigation, creating a storytelling experience inspired by modern streaming platforms.

---

# ✨ Features

- 🌌 Galaxy-themed landing page
- 🎬 Streaming-inspired home page
- 📖 Seven memory chapters
- 📸 Photo gallery for every chapter
- 🎵 Background music
- ✨ Smooth animations
- 💖 Responsive design
- 🎂 Birthday finale page

# 🚀 How to Run

1. Download or clone the project.
2. Extract the project folder.
3. Place all photos inside:

```
assets/images/
```

4. Place the background music inside:

```
assets/music/
```

5. Open:

```
index.html
```

or run the project using the **Live Server** extension in Visual Studio Code.
